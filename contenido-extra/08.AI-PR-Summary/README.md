# 08. IA para resumir Pull Requests

## Objetivo

Entender cómo la IA puede ayudar a revisar Pull Requests sin sustituir la revisión humana.

---

## Concepto clave

GitHub Copilot puede ayudar a explicar cambios en Pull Requests y generar resúmenes para revisores.

Este ejercicio no requiere crear un workflow. Está pensado para practicar productividad y revisión asistida.

---

## Ejercicio

1. Crea una rama nueva.
2. Modifica varios ficheros.
3. Abre una Pull Request.
4. Usa GitHub Copilot para resumir los cambios.
5. Compara el resumen generado con tu propio resumen manual.
6. Ajusta la descripción final de la Pull Request.

---

## Checklist de revisión humana

Comprueba manualmente:

- si el resumen es fiel a los cambios,
- si omite riesgos importantes,
- si exagera el alcance del cambio,
- si hay tests asociados,
- si el cambio tiene impacto en seguridad.

---

## Discusión

La IA puede acelerar la revisión, pero no debe aprobar código automáticamente sin criterio humano.
